# Simple Todo App

### Getting Started

- Clone the app
```
git clone https://github.com/Zerodayu/todo-app.git
```
 
- install dependencies 
```
# backend directory
cd backend
npm i
 
# frontend directory
cd frontend
npm i 
```

- add environtments
> open the `env.sample` then create `.env` file on the root directory and copy the sample format
> set the urls to the correct usage either `localhost:3000` or `localhost:3001`

- run the app
```
# backend directory
cd backend
npm run dev
 
# frontend directory
cd frontend
npm run dev
```
